# echoText
echoText is a **text-to-speech tool** that reads any text out loud. Designed for learners, readers, and multitaskers, it supports web, desktop, and can evolve into a browser extension.

## Features
- Convert text to speech in real time
- Adjustable voice speed and pitch
- Simple and intuitive interface
- Can be extended for web, desktop (Electron), or browser integration
- Supports multiple languages (future)

## Tech Stack
- **Frontend:** React (or vanilla JS for MVP)
- **Backend:** Node.js + Express (optional for storing user preferences)
- **Desktop / Extension:** Electron (for cross-platform desktop app)
- **Text-to-Speech:** Web Speech API / TTS libraries
